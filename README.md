# Web Camera SNS

スマートフォンのカメラを使って写真を撮影し、X(Twitter)に投稿できるWebアプリケーションです。

## 機能

- 📷 スマートフォンのカメラを使った写真撮影
- 🎯 シャッターボタンで簡単撮影
- 🐦 X(Twitter)への投稿（自動ハッシュタグ付き）
- 💾 撮影した写真のダウンロード
- 📱 モバイルフレンドリーなUI

## 技術スタック

- Vite
- React
- TypeScript
- GitHub Pages

## 使い方

1. アプリにアクセス
2. 「カメラを起動」ボタンをクリック
3. カメラへのアクセスを許可
4. 「シャッター」ボタンで撮影
5. 「ダウンロード」または「X(Twitter)に投稿」を選択

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## デプロイ

このプロジェクトはGitHub Pagesに自動デプロイされます。`main`ブランチにプッシュすると自動的にビルド・デプロイが実行されます。

## 注意事項

- HTTPSまたはlocalhostでのみカメラへのアクセスが可能です
- X(Twitter)への画像の直接アップロードはWeb Intent APIではサポートされていないため、ユーザーは撮影した写真をダウンロードして手動で添付する必要があります
